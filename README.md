## 📌 개요

Java 기반 서버사이드 웹 개발 기초를 다지기 위한 학습 레포지토리입니다.  
JSP/Servlet의 동작 원리부터 MVC 패턴을 적용한 쇼핑몰 프로젝트까지 단계적으로 학습합니다.

---

## 🛠 기술 스택

| 분류 | 기술 |
|------|------|
| Language | Java |
| View | JSP, HTML, CSS, JavaScript |
| Server | Apache Tomcat |
| Database | MySQL |
| DB 연동 | JDBC, Connection Pool (DBCP) |
| 패턴 | MVC Pattern |

## 🎯 학습 목표

- [ ] JSP/Servlet 기반의 서버사이드 렌더링 방식과 요청·응답 흐름 이해
- [ ] JDBC를 직접 다루며 ORM 없이 데이터베이스를 연동하는 방식 습득
- [ ] MVC 패턴을 적용한 쇼핑몰 프로젝트 완성
- [ ] Spring MVC 학습을 위한 Servlet/JSP 레이어 기초 확립
